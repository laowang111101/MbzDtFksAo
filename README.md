# 学校教学管理平台 java442

## 项目概述

学校教学管理平台是一个基于SSM（Spring + SpringMvc+ Mybatis）和Vue的前后端分离项目。该项目采用MVC三层体系结构，实现了基于B/S结构的Web系统，界面简洁，操作简单。

## 技术栈

- **后端**: JavaEE, Spring, SpringMvc, Mybatis
- **前端**: Vue
- **构建工具**: Maven
- **数据库**: MySQL

## 功能模块

（具体功能模块根据实际项目文档整理，以下为示例）

- 学生管理
  - 学生信息管理
  - 成绩管理
- 教师管理
  - 教师信息管理
  - 排课管理
- 课程管理
  - 课程信息管理
  - 课程资源管理

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- **操作系统**: Windows/Linux/MacOS
- **数据库**: MySQL 5.7+
- **JDK版本**: 1.8+
- **Web服务器**: Tomcat 8+

## 部署步骤

1. 准备好运行环境，安装相应的数据库和JDK版本
2. 导入项目数据库脚本
3. 使用Maven构建项目
4. 部署到Web服务器
